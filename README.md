# Want to run it locally on your machine?

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MiharisoaRakotonirina/Portfolio.git
   ```
2. **Install dependencies:**

   ```bash
   pnpm install
   ```

3. **Run the development server:**
   ```bash
   pnpm run dev
   ```
